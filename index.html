Flight Price Prediction

This project focuses on predicting flight ticket prices based on multiple factors such as airline, date of journey, departure and arrival times, duration, number of stops, and route. Customer Spending Prediction Using Machine Learning

This project analyzes customer behavior in an e-commerce platform to predict yearly spending using linear regression. Key factors such as time spent on the app, time spent on the website, session length, and membership duration are examined to determine their impact on spending.

Drug Classification

The goal of this project is to predict the type of drug a patient should be prescribed based on various medical attributes using a Decision Tree Classifier. The model is trained and evaluated on a dataset containing patient information such as age, sex, blood pressure levels, cholesterol levels, and sodium-to-potassium ratio.
